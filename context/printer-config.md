# Printer Configuration

This file stores the discovered or user-configured printer settings.
Edit these values or run `/find-printer` to auto-detect.

## Connection

| Setting | Value |
|---|---|
| Printer IP | `10.0.0.72` |
| IPP URI | `ipp://10.0.0.72:631/ipp/print` |
| CUPS Name | `HP-DeskJet-5200` |

## Capabilities

| Feature | Value |
|---|---|
| Color modes | `auto`, `monochrome`, `color`, `process-monochrome` |
| Duplex | Yes (long-edge, short-edge) |
| Paper loaded | A4 |
| Print speed (color) | 6 ppm |
| Scanner protocol | eSCL (AirScan) |

## Ink Cartridges

| Cartridge | Type |
|---|---|
| Tri-color | CMY combined ink cartridge |
| Black | Separate black ink cartridge |
